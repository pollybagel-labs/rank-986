## Headline findings

### 1. 986porsche.com is wrong about several names

| 986porsche claim | Likely correct (per PaintScratch + ATU) | Code |
|---|---|---|
| Lagoon Green Metallic | **Dark Teal Metallic** | M6W |
| Carmona Red Metallic | **Carmon Red Metallic** | M3W / 8B1 / 8B2 |
| Iris Blue Pearl | **Irisblau Pearl** (German) / Iris Blue Metallic (English) | 39V / 39N |
| Jungle Green Metallic | **Rainforest Green Metallic** (this was the canonical English name; "Jungle" appears to be a 986porsche translation) | 2A1 / 2A2 |
| Black Metallic | likely **Basalt Black Pearl** (older code) | 738 / 744 |

PaintScratch's **2002** page is itself inconsistent — it calls C9Z "Black Magic Pearl Metallic" and B9A "Candy White", names that no other source uses. Treating 2002 PaintScratch as an outlier; subsequent years' PaintScratch pages use canonical names ("Basalt Black Metallic", "Carrara White").

### 2. Codes that point to multiple colors across sources

- **M5W**: ATU 2000 says Arena Red; ATU 2001/2003/2004 + PaintScratch say Lapis Blue. Likely ATU 2000 is wrong, or M5W was reassigned. Other sources never associate M5W with red, so I'm inclined to drop the Arena Red M5W as an error.
- **3A8/3A9**: PaintScratch 2001/2002 = Lapis Blue Metallic. ATU 2004 lists "3A8, 3A9, LM5W, M5W = Lapis Blue Metallic" — which means **3A8/3A9 and M5W are aliases for the same color** (Porsche reissued the code).
- **23F**: ATU 2000 ambiguously lists this under both Slate Gray AND Lapis Blue. Probably an ATU listing error.
- **B9A**: Carrara White (canonical) vs Candy White (PaintScratch 2002 only).
- **C9Z**: Basalt Black Metallic (canonical) vs Black Magic Pearl (PaintScratch 2002 only).

### 3. ATU's late-986 listings appear contaminated with 987-era colors

ATU's 2003 and 2004 Boxster pages list **Crystal Silver Metallic (A7W)**, **Sand White (R9A)**, **Malachite Green Metallic (2B5)**, and **Jarama Beige Metallic (M1W)** — but PaintScratch shows these as 2005+ (987) colors. They do NOT appear on PaintScratch's 2003 or 2004 pages. Nor do they appear on the Porsche 986 Wikipedia article or any 986 enthusiast forums I checked.

**Conclusion: these are ATU catalog cross-pollination, not real 986 options.** Don't include them.

PaintScratch 2004 also lists **Dark Sea Blue (A5G)** which doesn't appear elsewhere — possibly a special-order or regional code, possibly an error. Flagging as unconfirmed.

### 4. The 2004 50 Jahre 550 Spyder Anniversary Edition

Per Stuttcars: only **GT Silver Metallic** (M7Z). 1,953 cars built, 500 for US market.
ATU 2003 lists GT Silver as a regular 2003 option, but other sources only show it for the 2004 LE. Treating 2004-LE-only as canonical.

### 5. Early years (1997–1999) have weak triangulation

PaintScratch doesn't cover 1997–2000. ATU's 1997 page lists only 4 colors (almost certainly incomplete). For pre-2000 colors I'm relying primarily on 986porsche.com + scattered forum confirmations. **Confidence on early-year color names and codes is lower.**

## Methodology

Three independent source TYPES selected for triangulation:
1. **Paint code authority / vendor catalog** — PaintScratch.com, AutomotiveTouchup.com (commercial — must match codes to real products)
2. **Enthusiast archive** — 986 Forum gallery, owner forums (real-world owner verification)
3. **Editorial reference** — 986porsche.com, Stuttcars (curated narrative)

This combination catches different failure modes:
- Vendors catch phantom colors or wrong codes
- Enthusiasts catch colors that exist in the wild but aren't well-documented
- Editorial sources catch context (special editions, year-specific options)

When sources disagree, multi-vendor consensus (PS + ATU agreeing) outranks single editorial claims.
